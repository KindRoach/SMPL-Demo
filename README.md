# Download SMPL model

Download models from [official site](https://smpl.is.tue.mpg.de/download.php) and put them under path ```models/smpl```.

```text
models
└── smpl
    ├── SMPL_FEMALE.pkl
    ├── SMPL_MALE.pkl
    └── SMPL_NEUTRAL.pkl
```

# Install Dependency

```bash
pip install -r requeirment.txt
```
